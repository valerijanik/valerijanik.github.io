Website for KlubOk ry

Creating a website for a non-commercial organisation

Tech

Wordpress, Adobe Xd, Photoshop, Illustrator

Role

Web Designer

Link to the website

KlubOk ry website

KlubOk ry is a non-commercial organisation, which aims to support Russian speaking families and mothers in Helsinki capital region.

But the time when I've joined them as a Graphic Designer volunteer, the organisation had a website build on Joomla, so for making our lives easier, we've moved it to Wordpress and fully redesigned it.

I've created all art assets that you can see there as well as the whole design. Right now we are working on Finnish and English versions of the website.
