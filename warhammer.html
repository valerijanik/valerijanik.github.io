Game based on Warhammer 40,000

This was a multi-player game with an aim to get licence from Games Workshop

Tech

Unity and C#, Git, Adobe Photoshop, Illustrator, Xd, Discord for team chatting and management, Trello for team management

Role

UI artists, UX designer and programmer

Link to repository

GitHub

My first project in semi-professional team

This game was my summer internship as a part of my studies at MediaStadi. I found those guys online and applied for a position of a developer and artist. So we've agreed that during the summer I'll complete design, art and implementation of Main Menu and in Game UI.

Because it was an online collaboration, we worked via GitLab and GitKraken (later swithed to Sourcetree). Well, I have to say I don't know it perfectly, but well enough to collaborate without any problems.

So during June and July I have used Adobe Xd to develop a prototype of main menu UI. During this time I've worked with Lead Designer and Project Lead to make sure I include every screen that they need for the first prototype build.

Then I've started working in Unity, implementing this prototype from Adobe Xd. This took a while at first to imagine how it should work, in what sequence and order. And then it became a repetitive job to make sure that every single button works and all sequences are connected. Also I've tried to make my code as clean as possible for the next person to have no problems understanding what is where and how it works.

After that I've started making art for UI. I've created buttons for main menu and two text boxes. August I've spend working in Unity, creating the UI system and it's flow based on the design from before. Also I've created several posters (art was done, I've only created layout, design and style) in Illustrator.
